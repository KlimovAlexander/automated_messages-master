===================================
|| Первичная настройка программы ||
========================================================================================================================
Перед первым запуском программы необходимо сделать следующее:

1. Загрузить необходимые для работы программы python библиотеки следующими командами в терминале:
   pip install oauth2client
   pip install googleapiclient
   pip install httplib2
   pip install email
   pip install selenium
   pip install BeautifulSoup4
   pip install pymysql
   pip install telethon
   pip install pyTelegramBotAPI
   pip install schedule

2. Ввести корректные данные в файл config.py в папке additional_functions (интегрирована база данных MySQL). Там, где
   указан комментарий correct data - это значит, что эти поля имеют корректную конфигурацию. Ее можно заменить,
   в случае необходимости. Там, где указан комментарий protect data - это значит, что эти данные менять нельзя

3. Запустить скрипты telegrambot.py в папке telegram
   Скрипт telegrambot.py отвечает за работу бота, который оповещает о входящих сообщениях, позволяет отвечать на них и
   регистрирует пользователей и менеджеров в базе данных.
   *** После этого этапа менеджеры уже могут регистрироваться в боте ***
   !!! Скрипт telegrambot.py должен работать 24/7 !!!
   Ссылка на бота: http://t.me/automated_messages_buldog_bot

4. Запустить скрипт settings.py в папке telegram для создания таблиц в базе данных

5. Запустить скрипт settings.py в папке telegram для добавления сессий пользователей telegram, которые участвуют в
   рассылке. После того как вы добавите сессии всех пользователей в папке telegram появятся файлы с расширением
   ".session". Все эти файлы необходимо перенести в папку sessions. Один профиль необходимо оставить для мониторинга
   отчетов. При запросе телефона профиля на русском, в его случае необходимо ввести не телефон, а название "statistic".
   После регистрации профиля его необходимо поместить в папку statistic

6. Запустить скрипт report.py. Этот скрипт также, как и скрипт telegrambot.py, должен работать 24/7 - он отвечает за
   мониторинг и оповещения об отчетах

!!! Порядок действий важен !!!
*** Данную процедуру необходимо провести один раз после загрузки проекта на сервер ***

==============================
|| Использование программы  ||
========================================================================================================================
Программа выполняет 3 основные функции:
1. Рассылка по электронной почте (gmail)
2. Рассылка в социальной сети LinkedIn
3. Рассылка в Telegram (+ мониторинг ответных сообщений от клиентов через телеграм бота)

Запуск рассылок:
1. Запуск рассылки по электронной почте.
Чтобы сделать рассылку по электронной почте, необходимо запустить скрипт automated_email.py в папке automated_email.
Все остальное программа сделает за вас

2. Запуск рассылки в LinkedIn.
Чтобы сделать рассылку в LinkedIn необходимо, запустить скрипт linkedin.py в папке linkedin
Все остальное программа сделает за вас

3. Запуск рассылки в Telegram
В случае с Telegram запуск рассылки также прост - необходимо запустить скрипт auto_messages.py в папке telegram. Однако
стоит учитывать момент, что программа также имеет функционал по мониторингу ответов от клиентов. Этот функционал
реализован в скрипте telegram.py. Но в силу технических особенностей, рассылка и мониторинг не могут работать
одновременно, поэтому при запуске рассылки в Telegram необходимо выключать скрипт telegram.py. А после окончания
рассылки включать его обратно
!!! Напоминаю, что скрипт telegrambot.py должен работать постоянно !!!

================================================
|| Как изменить текст сообщений в рассылке ?  ||
========================================================================================================================
На момент сдачи проекта текст сообщения в рассылке соответствует тому, который был дан при разработке проекта. Однако,
если вам необходимо поменять текст сообщений, вы можете сделать это следующим образом:

1. Текст сообщения в рассылке по электронной почте находится в файле "Сообщение.html" в папке automated_email.
Если вам необходимо изменить текст сообщения в рассылке по электронной почте, достаточно просто изменить файл
"Сообщение.html" так, как вам нужно.
Тема сообщения в рассылке по электронной почте находится в файле "Тема.txt" в той же папке. Чтобы изменить тему
сообщения в рассылке по электронной почте, нужно аналогично изменить текст в файле "Тема.txt" на нужный

2. Текст сообщения в рассылке в LinkedIn находится в файле "Сообщение.txt" в папке linkedin. Чтобы изменить текст
сообщения в рассылке в LinkedIn, вам необходимо изменить текст в файле "Сообщение.txt". Но учтите, что в случае с
LinkedIn есть техническое ограничение на смайлики в тексте. То есть в файле "Сообщение.txt" не должно быть смайликов.
Также стоит учитывать, что у LinkedIn есть ограничение в длине сообщения в размере 300 символов. Если вы введете более
300 символов, то отправятся только первые 300 символов сообщения. Поэтому будьте с этим внимательны

3. Текст сообщения в рассылке в Telegram находится в файле "Сообщение.txt" в папке telegram. Если вам необходимо
изменить текст сообщения в рассылке в Telegram, вам нужно изменить текст в фале "Сообщение.txt"
